# About
This project was for me to use HTML and CSS in a practical way, so I decided to make a website mockup for my sister's photography business! I wanted to practice what I have learned and have more freedom with the project and to learn more techniques as I went. 

While overall relatively simple, I tried to add some more complex elements like the contact form and hero image. I wanted to have organized pages with clean formatting. CSS is still a bit new to me, so I wanted to focus on making my stylesheet readable, and experimenting to see how the page would change to get a better understanding of what I was writing.
